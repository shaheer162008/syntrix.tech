# Nexiler - Brand Kit & Design System

## Colors

### 1. Primary Accent (Purple)
**Hex:** `#a855f7`
- Used for: Brand Logo ("nexiler"), Call to action buttons, prominent text highlights, icon borders, and main hover states.
- Ensures the premium touch and visually strikes a contrast against dark backgrounds.

### 2. Background Base (Dark Grey)
**Hex:** `#1B1B1B`
- Used for: Primary background (`--background`), Card backgrounds, Navbar/Footer backgrounds, and the Fill color of the Globe Wireframe.
- Represents the core premium dark theme of the website (Replaced pure blacks).

### 3. Text & Highlights
**Hex:** `#ffffff`
- Used for: Headings, primary content, glowing overlays.
- Variants: `rgba(255,255,255,0.9)` down to `0.05` for borders, dividers, subtle texts.

**Hex:** `#999999` (or `oklch(0.708 0 0)` Tailwind Muted)
- Used for: Body text, sub-descriptions, secondary navigation items.

## Typography
- **Primary Font:** Geist Sans (system-ui, sans-serif)
- **Code & Tech Specs:** Geist Mono (ui-monospace)
- **Logo Font Mapping:** The logo strictly uses `.font-negan` mapped in CSS.
